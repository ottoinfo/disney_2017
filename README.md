# Disney Code Challenge

What I am using
- Webpack 3.0
- React & Redux( Sagas )
- React Router 4
- Redux-CLI (Blueprints)

## Setup

```
npm i
npm run dev
```

### [Redux-CLI](https://github.com/SpencerCDixon/redux-cli)

Quick way to get components built w/ Tests

Install 

```npm i redux-cli -g```

Example Use

```redux g dumb MyComponent```
